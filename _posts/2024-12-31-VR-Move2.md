1. Teleportation 이란?

- Teleportation : 오른손 컨트롤러의 광선을 직선에서 곡선으로 바꾸기
- 텔레포테이션이란? 현재 위치에서 지정한 위치로 즉각 이동하는 기능으로서 직선이나 곡선으로 라인을 표시하며 목표점으로 이동하도록 한다. 유니티에서 제공하는 텔레포테이션 방법에는 두가지가 있다. Teleportation Area 방법과 Teleportation Anchor 방법이다.
  - Teleportation Area : 모든 곳으로 이동할 수 있는 기능
  - Teleportation Anchor : 설정한 Anchor의 위치와 방향으로 움직이도록 한다.
- 두 가지 방법 모두 Locomotion System과 Teleportation Provider가 있어야 구현이 된다. 어떤 시나리오를 선택하여 각각 기능을 활성화 비활성화 하면서 이동을 하게 되면 다양한 게임 속 연출이 가능하게 된다.

1. Right Controller 활성화 (XR Direct Interactor로 만든 Custom Right Controller 말고 기존에 기본적으로 제공해주는 Right Controller를 의미한다, 왜냐하면 기본 Right Controller 에는 Line Renderer 등이 들어 가 있어서 Teleportation 할 때 사용할 수 있음)
2. Right Controller → XR Ray Interactor → Line Type 을 Projectile Curve 로 바꾼다.

2. XR Origin→ XR Origin → Tracking Origin Mode → Device 로 설정함

모드가 "Device"일 때 XR 런타임은 일반적으로 추적된 장치의 위치를 고정된 공간에서의 위치로 보고합니다. 예를 들어 HMD(Head-Mounted Display)가 시작될 때의 초기 위치와 동일시하게 셋팅한다. 이 모드에서는 XR 원점의 "Camera Y Offset"을 설정하여 Main Camera를 원하는 높이로 지상 위에 배치할 수 있습니다. 모드가 "Floor"일 때 XR 런타임은 일반적으로 추적된 장치의 위치를 플레이어의 실제 바닥과 관련하여 보고합니다. 이 모드에서는 Unity가 카메라 오프셋의 높이를 자동으로 지우는데, 이는 추적 원점을 인위적으로 높이 올릴 필요가 없기 때문입니다. "Not Specified"로 모드를 설정하면 XR 런타임의 기본 모드를 사용합니다.
